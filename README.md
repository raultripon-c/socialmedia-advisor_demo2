# Social Media Advisor Demo 2

Standalone Social Media Advisor preview (Campaigns, Content Board, Amplify).

**Live:** https://raultripon-c.github.io/socialmedia-advisor_demo2/

## Run locally

```bash
npm install
npm run dev
```

Opens at http://localhost:4320

## Routes

- `/campaign-studio/campaigns`
- `/campaign-studio/campaigns/new`
- `/campaign-studio/campaigns/workspace/:id`
- `/campaign-studio/campaigns/:id/dashboard`
- `/campaign-studio/content-board`
- `/campaign-studio/amplify-preview`
